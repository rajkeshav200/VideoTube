# 🎥 VideoTube

A full-stack YouTube-inspired video sharing platform built with the MERN stack. 
Users can upload and stream videos, create playlists, track watch history, search by name
interact through likes, comments, Tweets and subscriptions, while enjoying secure 
authentication and cloud-based media storage.

---

## 🚀 Features

- **Secure Authentication:** JWT authentication using HTTP-only cookies.
- **Video Uploads:** Upload videos and thumbnails using Cloudinary & Multer.
- **Video Streaming:** Stream videos with optimized cloud storage.
- **Video Search:** Search videos by title and keywords.
- **Filtering & Pagination:** Browse content efficiently with server-side pagination and filtering.
- **Playlists:** Create, update, and manage personal playlists.
- **Watch History:** Automatically track previously watched videos.
- **Channel Management:** Create and manage user channels.
- **Likes & Comments:** Like videos and engage through comments.
- **Subscriptions:** Subscribe and unsubscribe from creators.
- **User Dashboard:** Manage uploaded videos, playlists, profile, and account settings.
- **Protected Routes:** Secure role-based access using JWT.
- **RESTful APIs:** 15+ APIs built following the MVC architecture.

---

## 🛠️ Tech Stack
**Frontend:** React.js, React Router, Context API, Tailwind CSS.
**Backend:** Node.js, Express.js, MongoDB, Mongoose.
**Authentication:** JWT, HTTP-only Cookies.
**Media Storage:** Cloudinary, Multer.
**API Testing:** Postman.

## 📖 How It Works

1. Register and log in securely.
2. Upload videos and thumbnails.
3. Browse, search, and filter videos.
4. Stream videos with automatic watch history tracking.
5. Create and manage playlists.
6. Like, comment, and subscribe to creators.
7. Manage uploaded content and channel settings from the dashboard.
